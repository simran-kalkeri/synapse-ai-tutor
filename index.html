Core Subsystem Architecture

🔍

GraphRAG Retrieval

Hybrid knowledge graph-expanded vector retrieval. Identifies concept vertices from student query nodes, extracts 2-hop prerequisite links using NetworkX, and boosts chunk results.

📈

Adaptive Assessment

Custom 15-question diagnostic covering Beginner, Intermediate, and Advanced tiers. Determines initial proficiency profiles and dynamically detects local knowledge gaps.

🎙️

Local Voice Layer

Provides zero-friction hands-free learning. Powered by Faster-Whisper local voice speech-to-text and ElevenLabs or gTTS speech synthesis.

🪟

Frosted Glassmorphism Theme

Premium dark theme featuring a tailored HSL dynamic color scheme, outfit geometric headers, and dynamic animations designed for modern technical products.

🗺️

Offline Cytoscape Roadmaps

Generates beautifully structured tree layouts for personalized studies. Features completely self-contained local copies of cytoscape.js and dagre layout plugins for 100% offline portability.

🎬

Visual Animation Engine

Illustrates intricate structures (RAG flows, attention weights, recursive stack trees) step-by-step with synchronized audio narration and PIL crossfades.
